Pharmadika

Fullstack Bootcamp Project

The features I work on this E-commerce Web App project:

Categories at home page will link to product list with category selected.

New Products at home page shows four latest products added in the app.
Product Card shows name, price, and stock for each product.

Product List shows all products with:
- Filter; Category & Bottle Capacity.
- Sort.
- Search.
- Pagination.

Admin Dashboard presents summary of:
- Number of Sales.
- Revenue with its chart (based on selected dates).
- Profit with its chart (based on selected dates).
- Cost with its chart (based on selected dates).
- Top 3 products.

Transaction shows all transactions and the details with approval mechanism:
- Red (rejected).
- Green (approved).
- White (not yet confirmed).

Report shows all approved transactions and the details.

​​​​​​​

The technologies I use on this project:
- React.js.
- Sweetalert.
- Material UI.
- Styled Component.
- Axios.
- Node.js.
- Express.js.
- MySQL.
- ORM Sequelize.
